Lesson 7: Screen Curtain

A handy tool to save battery life as well as practice navigating an iPad without seeing the screen is called Screen Curtain. Screen Curtain turns off the screen display, while still allowing you to operate the ipad using VoiceOver gestures.

Screen Curtain: Three-Fingers (3 Taps)

To turn Screen Curtain on and off, use three fingers together and tap three times quickly.

Try the gesture now. Your screen will go completely black, but VoiceOver will keep talking! Perform the exact same three-finger triple tap again to turn the screen back on.

Note: Check the information below if this gesture is not working.

Troubleshooting: Zoom Feature Conflict (4 Taps)

If you try to use the triple tap and Screen Curtain is not working, this is usually because you also have the Zoom feature active.

When both VoiceOver and Zoom are active at the same time, the iPad changes the Screen Curtain shortcut so they do not clash. To turn Screen Curtain on or off in this mode, you must use three fingers and tap four times quickly.

Tip: Use three-finger double tap to turn on and off the magnifying tool when the Zoom feature is on.
