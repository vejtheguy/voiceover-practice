Lesson 4: Selecting Items on the Page

With VoiceOver, there are various ways to select items on the page.

Swipe: Select by Swiping

Using one finger, swipe to the right to move to the next item down the page. Using one finger, swipe to the left to move to the item back up the page.

Swiping is a great way to navigate up and down a webpage when scrolling is not working effectively.

Try swiping up and down through the three items below.

This is a HTML h3 tag with text.

Drag: Select by Dragging

Drag one finger slowly around the screen. VoiceOver will read whatever is underneath your finger without activating anything.

Tap: Select by Single Tap

Using one finger, tap items on the page. On a webpage, this could be things such as text, images, or buttons.
