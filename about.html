About Learning Digital Accessibility

Welcome to my project website! I created this space as a personal sandbox to practice web development and, more importantly, to directly apply everything I am learning about digital accessibility to the site itself as I build it.

Behind the Project

During the summer of 2026, I received the opportunity to join the internship program through the American Foundation for the Blind (AFB), and explore the world of web accessibility. This website serves as a personal documentation hub where I can organize my notes, track how I am utilizing WCAG criteria, and experiment with accessible web design through manual coding.

The Goal of This Project

Instead of keeping everything I have been learning to myself, I am using this project to create practical, hands-on lessons for how to use assistive technologies. My goal is to build helpful guides for beginners to learn how to use VoiceOver, TalkBack, and NVDA. By taking the time to create these guides, I am not only reinforcing my own knowledge of these tools, but also hoping to provide a resource for others who may be new to using assistive technologies.
